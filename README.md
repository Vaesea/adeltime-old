# Adel Time
A horrible "volcanic penguin" named Volcanic Penking is up to something bad! He's doing stuff you'd expect from a guy with a head that looks like an egg and you must help Adel stop him. You don't even know him? Well too bad! You don't even know who Adel is? Well, she's a random penguin from a cold, cold place named Antarctic Snow.

## Info
This is inspired by Turrican II, Sonic 2's Scrapped Time Travel story and Super Mario Bros. There's 3 things you thought you'd never hear in the same sentence all together...

## Compilation Instructions
1. Get Haxe 4.3.7
2. Get HaxeFlixel (including Flixel-Addons)
3. Get hxcpp (needed for windows builds)
4. Get Visual Studio Community (install the c++ stuff, like the windows 11 sdk and the msvc stuff) (i know, this installs around 7 GB of stuff, but it's needed i think)
5. Get Polymod (including thx.core, thx.semver, jsonpath and jsonpatch)
6. Use git version of thx.core (haxelib git https://github.com/fponticelli/thx.core)
7. Compile using "lime test (platform) -debug" (You can use -release or whatever, but I recommend against that right now)
8. You should be done. If there's anything I missed please let me know.
